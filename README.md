## App de produtos

App de delivery de produtos onde serão expostas todas categorias de produtos.

### Tecnologias

- nodejs
- express
- handlebars
- body-parser
- connect-flash
- express-session
- mongodb(dockerizado)
- docker
- fontawesome
- bootboxjs
- google fonts
